# :copyright: What is the brand?

A Memory Card game made with pure javascript 💜

## 🛠️ Stack

- PUG
- SASS
- Pure javascript
- Webpack 4.0 to transpile SASS into CSS, Pug into HTML and JS ES6 to ES5
- Jest to automated tests (91.46% of coverage)
